# Ипотека24.KZ — заметки по проекту

Контекст для продолжения работы (в т.ч. на новом компьютере). Сам сайт описан в [README.md](README.md).

## Что это
Статический информационный портал об ипотеке в Казахстане (HTML/CSS/JS, без сборки).
Хостинг — GitHub Pages, домен **ipoteka24.kz** (через hoster.kz).

- Репозиторий: https://github.com/artempomelov/ipoteka
- Живой сайт: https://ipoteka24.kz
- Резерв: https://artempomelov.github.io/ipoteka/

## Структура
```
index.html     — главная: hero, гайд-тизер, банки, калькулятор, оценка шансов,
                 eGov-блок, застройщики, новости-тизер, правовой CTA, форма обратной связи
guide.html     — полный гайд (6 шагов)
legal.html     — правовые основы: памятки + законы со ссылками на adilet.zan.kz
news.html      — новости ипотеки (10 шт)
prequal.html   — отдельная страница оценки шансов (с шарингом результата по ссылке)
bank.html      — страница банка (bank.html?bank=<slug>): программы + калькулятор
css/styles.css — стили (светлая тема в цветах флага РК: голубой #00afca, золотой #fec50c)
js/data.js     — ВЕСЬ КОНТЕНТ: BANKS, DEVELOPERS, NEWS, LAWS, LEGAL_GUIDES, STEPS + хелперы логотипов
js/calculator.js — переиспользуемый аннуитетный калькулятор (со страхованием)
js/prequal.js  — оценка шансов (LTV, КДН по АРРФР с прожиточным минимумом, лимиты по городам)
js/app.js      — рендер главной, навигация, форма обратной связи, кнопка Telegram, eGov
js/pages.js    — рендер guide/legal/news
assets/logos/  — логотипы банков (<slug>.png/svg) и застройщиков (dev-<slug>.png)
worker/feedback-worker.js — код Cloudflare Worker (форма → Telegram)
serve.ps1      — локальный статический сервер для предпросмотра
```

## Интеграции (живут в облаке, не в репозитории)
- **Форма обратной связи → Telegram** через Cloudflare Worker.
  - Endpoint в `js/app.js`: `FEEDBACK_ENDPOINT = https://snowy-term-fb52.artem-pomelov.workers.dev`
  - Секреты воркера (в Cloudflare): `BOT_TOKEN`, `CHAT_ID=95507672`. Белый список доменов — в `ALLOWED_ORIGINS` воркера. С localhost не работает (CORS — by design).
- **Telegram-бот** @ipoteka24kz_bot. Кнопка «Написать в Telegram» в `js/app.js`: `TELEGRAM_URL`.
- Запасной путь формы (если endpoint пуст) — отправка через mailto на `FEEDBACK_EMAIL`.

## Ключевые цифры/правила (актуализированы под июнь 2026)
- Базовая ставка НБ РК — 17% (с 5 июня 2026).
- С 1 июля 2026: потолок ГЭСВ по ипотеке — 20% при взносе от 30%, до 25% при меньшем.
- Рыночные ставки банков и дефолты калькулятора отражают это.
- Прожиточный минимум для КДН — `SUBSISTENCE_MIN` в `js/prequal.js` (ориентир ~46228 ₸).
- Лимиты госпрограмм по городам — `PQ_CITIES` в `js/prequal.js`.

## Рабочий процесс
- Правки контента — в `js/data.js`. Превью — `serve.ps1` или открыть index.html.
- Любой `git push` в ветку `main` обновляет сайт на ipoteka24.kz за ~1 минуту (GitHub Pages).
