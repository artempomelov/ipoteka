# 🚀 Запуск проекта Ипотека24.KZ на новом компьютере

Эта папка — **полная копия проекта** с историей git. Перенеси её на новый компьютер
(через OneDrive, флешку или просто скопируй) и пройди шаги ниже.

> Краткое описание проекта — в [PROJECT_NOTES.md](PROJECT_NOTES.md).
> Документация сайта — в [README.md](README.md).

---

## Шаг 1. Установить программы

1. **Git** — скачай с https://git-scm.com → установи (всё по умолчанию, «Next»).
2. **Claude Code** — открой https://claude.com/code и поставь по инструкции
   (CLI, десктоп-приложение для Windows или расширение для VS Code — на выбор).
3. *(по желанию)* **GitHub CLI** — https://cli.github.com — упрощает вход в GitHub.
4. *(по желанию)* **Node.js** — https://nodejs.org — нужен, только если ставишь Claude Code через npm.

---

## Шаг 2. Проверить, что папка на месте

Эта папка должна лежать в удобном месте, например:
`C:\Users\<ты>\Documents\Ipoteka` (или где угодно — путь не важен).

Внутри должны быть: `index.html`, папки `js`, `css`, `assets`, скрытая папка `.git` и этот файл.

---

## Шаг 3. Запустить Claude Code в этой папке

- **CLI:** открой терминал (PowerShell), перейди в папку и запусти `claude`:
  ```powershell
  cd "путь\к\папке\Ipoteka"
  claude
  ```
- **Десктоп / VS Code:** просто открой эту папку как проект.

При первом запуске Claude попросит **войти в твой аккаунт Anthropic** — тот же, что и раньше.

После входа скажи мне: **«продолжаем проект Ипотека24, прочитай PROJECT_NOTES.md»** —
и я сразу войду в контекст.

---

## Шаг 4. Подключить GitHub (чтобы публиковать изменения)

Чтобы `git push` обновлял сайт, нужно авторизоваться один раз:

- Если поставил GitHub CLI:
  ```powershell
  gh auth login
  ```
  (выбирай: GitHub.com → HTTPS → Yes → Login with a web browser)

- Либо просто сделай первый `git push` — откроется браузер для входа.

Проверка, что репозиторий на месте:
```powershell
git remote -v
git pull
```
Должен показать `origin  https://github.com/artempomelov/ipoteka.git`.

---

## Шаг 5. Войти в облачные сервисы (настройки уже там, ничего настраивать заново не надо)

- **GitHub** — github.com (репозиторий и хостинг сайта).
- **Cloudflare** — dash.cloudflare.com (Worker для формы → Telegram).
- **Telegram** — бот @ipoteka24kz_bot (через @BotFather, если понадобится токен).
- **hoster.kz** — домен ipoteka24.kz (трогать не нужно, работает сам).

---

## Как всё устроено (быстрый ориентир)

- Правишь контент → файл `js/data.js` (банки, застройщики, новости, законы).
- Сделал `git push` в ветку `main` → сайт **ipoteka24.kz** обновляется за ~1 минуту.
- Локальный предпросмотр: запусти `serve.ps1` или открой `index.html` в браузере.

## Не забыть (по желанию, безопасность)
🔒 Перевыпустить токен Telegram-бота: в @BotFather команда `/revoke`, затем обнови
`BOT_TOKEN` в Cloudflare (Worker → Settings → Variables). Сайт менять не нужно.
