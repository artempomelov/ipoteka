# Ипотека24.KZ — портал об ипотеке в Казахстане

Статический информационный сайт: правовые основы, условия банков, калькулятор аннуитетного платежа и предложения застройщиков.

## Структура
```
index.html        — главная: гайд, законы, банки, калькулятор, застройщики
bank.html         — страница банка (открывается как bank.html?bank=<slug>)
css/styles.css    — стили (тёмная тема, адаптив)
js/data.js        — контент: законы, банки с программами, застройщики (редактируйте здесь)
js/calculator.js  — переиспользуемый аннуитетный калькулятор (со страхованием)
js/app.js         — рендер главной и навигация
js/bank.js        — рендер страницы банка и переключатель программ
serve.ps1         — лёгкий локальный статический сервер (для предпросмотра)
```

## Запуск
Откройте `index.html` в браузере. Сборка не нужна.

Локальный сервер (Windows, PowerShell):
```
powershell -ExecutionPolicy Bypass -File serve.ps1 -Port 5500
```
затем откройте http://localhost:5500

## Калькулятор и страхование
Калькулятор считает аннуитетный платёж. Галочка «Учитывать страхование» добавляет
≈0.5% от суммы займа в год (средний рыночный тариф ипотечного страхования в РК).
Значение настраивается константой `INSURANCE_ANNUAL_RATE` в `js/data.js`.

## Программы банков
Каждый банк в `js/data.js` имеет массив `programs`. У программы есть описание,
список «кому подходит», условия и блок `calc` — дефолтные параметры калькулятора,
которые подставляются при выборе программы (пользователь может их менять).

## Как обновлять данные
Все условия банков и застройщиков — в `js/data.js`. Меняйте массивы `BANKS`, `DEVELOPERS`, `LAWS`, `STEPS`.

> ⚠️ Данные о ставках и условиях собраны из открытых источников (2026) и приведены для ориентира. Перед публикацией сверяйте с официальными сайтами банков и нормативными актами на adilet.zan.kz.

## Публикация
Подходит для GitHub Pages: загрузите репозиторий и включите Pages для ветки `main` (корень).
